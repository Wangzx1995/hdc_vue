# 安装说明
1、请先安装Node.js环境
2、打开命令行工具，进入该项目目录
3、执行 npm install 安装依赖库
4、执行 npm run dev 启动项目，命令行会自动打开浏览器并访问 http://localhost:9528

# 安装依赖库过程中遇到的坑
1、使用NPM安装依赖库时，若无法访问官方npm的url
可在公司OA中申请官方npm的http访问地址的权限，或将npm的--path映射到国内淘宝的npm的地址，具体操作可问度娘
2、跟cache相关的错误提示，可通过下面的命令行来解决
npm cache clean --force
npm cache verify
3、启动项目过程中遇到permission类错误提示，是命令行权限问题
重新使用鼠标右键，选择“以管理员身份运行”方式打开命令提示符，重新进入项目目录启动项目

# 特别声明
1、请不要将 node_modules 目录提交到svn中，里面的文件超过10万个，会严重影响更新速度
